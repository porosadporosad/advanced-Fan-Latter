# Chapter 3-2 주특기 숙련 (React)
개인과제

## 🖥️프로젝트 소개
“그룹 아티스트 팬레터함”을 만들어 봅시다.

### 🕰️개발 시간
- 24.01.29 - 24.02.05

### ⚙️개발 환경
- React(18.2.0)
- react-router-dom(6.21.3)
- redux(5.0.1)
- react-redux(9.1.0)

### 📌주요 기능
- 팬레터 CRUD 구현
- 아티스트별 게시물 조회 기능 구현
- 원하는 아티스트에게 팬레터 등록 구현
- 팬레터 상세 화면 구현
- 상세화면에서 팬레터 내용 수정 구현
- 상세화면에서 팬레터 삭제 구현
